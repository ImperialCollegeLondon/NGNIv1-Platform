This directory contains a source code for CyUSB C# DLL.

PROJECT FILE
	Open CyUSB.csproj file in the Visual Studio.
	Please note, first time it will ask user to save project solution file.
	This project file is created using Visual Studio 2008. It can be migrated to higher version of Visual Studio.
	

PROJECT OUTPUT
	After project build, output files will be copied in the 'lib' directory.
	
	Release :lib\CyUSB.dll 
	Debug   :lib\Debug\CyUSB.dll

API GUIDE DOCUMENT
	Please refer Cypress CyUSB.NET Programmer's Reference document for API interface of this DLL.
	You can find this document after installing FX3 SDK installer under following directory. 
	Directory : Cypress\Cypress USBSuite\library\c_sharp\CyUSB.NET.pdf or CyUSB.NET.chm
